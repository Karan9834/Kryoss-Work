# WhiteLabelFox React Application

## Project Info

This project is a modern Single Page Application (SPA) built specifically with pure React.js and Vite.

## Technologies

- **Vite**: Ultra-fast frontend build tool
- **React.js**: Standard Javascript (No TypeScript) UI Library
- **Tailwind CSS v4**: Utility-first CSS framework natively integrated
- **Shadcn UI**: Unstyled accessible components
- **React Router v6**: Client-side declarative routing

## Getting Started

### Prerequisites

Ensure you have Node.js and npm installed on your machine.

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```

### Scripts

- `npm run dev`: Starts the local Vite development server
- `npm run build`: Compiles the application for production
- `npm run preview`: Previews the built production bundle locally
- `npm run lint`: Runs ESLint for code formatting checks
